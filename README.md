# mosquit-legal

This repository hosts the public legal, privacy, and support pages for the
**MosQuit** iOS app.

It is intended to be published with **GitHub Pages** from the `main` branch
root (no build step required).

## Pages

| File | Description |
|------|-------------|
| `index.html` | Marketing landing page |
| `privacy.html` | Privacy policy |
| `support.html` | Support page and FAQ |
| `style.css` | Shared CSS used by all pages |

## Deploying

1. Go to **Settings → Pages** in this repository.
2. Under *Source*, select **Deploy from a branch**.
3. Choose the `main` branch and the `/ (root)` folder.
4. Save — GitHub Pages will serve the site at your Pages URL within a minute.

## Design principles

- Plain HTML and CSS only — no JavaScript, no frameworks, no build system.
- No external fonts or assets.
- No tracking, cookies, or analytics.
- Responsive layout for mobile and desktop.
